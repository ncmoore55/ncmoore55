### JustinFloors Job Tracker
A web-based job management application built for a real flooring contractor business to replace disconnected spreadsheets, emails, and calendars with a single centralized platform.

- Designed and normalized a MySQL relational database schema, eliminating redundancy and ensuring data integrity across job, material, and scheduling records
- Built full CRUD operations for managing active jobs, material orders, delivery tracking, installation schedules, and QuickBooks document references
- Developed the user interface in Python using Streamlit, enabling real-time project status updates through a clean dashboard
- Applied database normalization principles to make the schema scalable and maintainable as the business grows

**Stack:** Python · Streamlit · MySQL · SQL
